- 术语修正（3000+ 规则）
- 智能断句拆分
- 音频波形时间轴

## 快捷键
- `Space` 播放/暂停
- `Ctrl+Z` 撤销
- `Ctrl+Y` 重做
- `Enter` 编辑字幕
- `Delete` 删除字幕
- `Ctrl+滚轮` 缩放时间轴
